Web Design

El desarrollo web hace referencia a la creación y mantenimiento de sitios web. Implica una amplia gama de acciones, desde la codificación y el diseño hasta la gestión de contenidos y la administración del servidor web. Asimismo, el desarrollo web consiste en utilizar lenguajes de programación como: HTML,CSS, JavaScript, y muchos otros, todo con el fin de escribir programas que harán que nuestro sitio sea mas práctico.
