Elementos metadata

Elemento Descripcion Documentacion
base Especifica la direccion URL base base
link Especifica la relación entre el documento actual y un recurso externo link
meta Metainformación. Sirve para aportar información sobre el documento. meta
scrip Añade un ejecutable usualmente se refiere a Java scrip script
style Es el elemento encargado de indicar la información de estilo. style
title Define el título del documento que se muestra en un navegador title

Elementos flow pt1

Elemento Descripcion Documentacion
ARTICLE Representa una composición auto-contenida en un documento o página article
B Indica que el texto debe ser representado con una variable bold, o negrita b
AUDIO Se usa para insertar contenido de audio en un documento HTML audio
MATH Es el elemento superior en MathML math
STRONG Apropiado para marcar con especial énfasis las partes más importantes de un texto. strong
BR Produce un salto de línea en el texto. Es útil para escribir un poema o una dirección br

Elementos sectioning

Elemento Descripcion Documentacion
ARTICLE Representa una composición auto-contenida en un documento o página article
ASIDE Representa una sección de una página que consiste en contenido que está indirectamente relacionado con el contenido principal del documento aside
NAV Representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, dentro del documento actual u otros documentos nav
SECTION Representa una sección genérica de un documento section

Elementos phrasing pt1

Elemento Descripcion Documentacion
BR Produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección br
LABEL Representa una etiqueta para un elemento en una interfaz de usuario label
SPAN Es un contenedor en línea. Sirve para aplicar estilo al texto o agrupar elementos en línea. span
SUP Define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño sup

AREA Sirve para crear las distintas secciones en las que se puede dividir un mapa de imágenes en el cliente. area
MAP Descripción de uno o dos párrafos map

Elementos embedded

Elemento Descripcion Documentacion
AUDIO El elemento audio se usa para insertar contenido de audio en un documento HTML o XHTML audio
CANVAS El elemento se puede utilizar para dibujar gráficos a través de secuencias de comandos canvas
SVG Contenedor para graficos SVG svg
PICTURE El elemento HTM se puede utilizar para dibujar gráficos a través de secuencias de comandos picture
IMG Sirve para insertar una imagen img

Elementos interactive

Elemento Descripcion Documentacion
MENU Es un elemento desaprobado. En su origen fue pensado para crear menús, en la actualidad es inutil. menu
OBJECT Representa un recurso externo, que puede ser tratado como una imagen, un contexto de navegación anidado object
INPUT Se usa para crear controles interactivos para formularios basados en la web con el fin de recibir datos del usuario input
