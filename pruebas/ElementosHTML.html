Elementos HTML
Elemento Descripcion Documentacion
HTML Es la raiz del documento Documentacion
Head Contiene informacion que no esta a plena vista respecto al documento Documentacion
Title Este etiqueta va contenida en head y su funcionalidad es indicar el nombre que se muestra en la pagina del buscador Documentacion
main Se usa para representar la informacion dominante dentro el body Documentacion
Div Es un contenedor generico que no tiene efecto en el contenido a menos que se le apliquen estilos Documentacion
p El elemento hace referencia a un parrafo, pueden ser vistos como bloques de texto Documentacion
input El es usado para recibir informacion del usuario, este varia dependiento el type, por ejemplo button, date, password, etc Documentacion
h1-h6 Representan jerarquia en titulos siendo h1 la mas alta y h6 la mas baja Documentacion
span span es un inline element a diferencia de div es usado comunmente en texto Documentacion
article es un elemento que contiene informacion independiente de la pagina Documentacion
section es un elemento que se usa como contenedor para un area de la pagina Documentacion
nav se usa para agregar enlaces ya sea dentro el documento o hacia otro Documentacion
footer etiqueta que contiene informacion en la parte inferior de la pagina, esta puede ser informacion de la pagina como autor, links, copyright, etc. Documentacion
aside Se usa para presentar informacion que no esta muy relacionada al topico general, usualmente se presenta a los lados de la pagina Documentacion
strong Sirve para indicar la importancia del contenido, usualmente el texto se renderiza en bold Documentacion
sup Se usa para renderizar un texto un poco mas arriba que otro, por ejemplo un exponente Documentacion
sub Se usa para renderizar un texto un poco mas abajo que otro Documentacion
br Se usa para agregar un salto de linea Documentacion
hr Se despliega una linea que se interpreta como cambio de tematica Documentacion
wbr Es una etiqueta que sirve para indicarle al navegador que puede insertar un salto de linea Documentacion
em Es una etiqueta que sirve para dar enfasis a un texto dado Documentacion
mark Es una etiqueta que sirve para resaltar un texto Documentacion
q Sirve para hacer una cita bibliografica referente al texto contenido Documentacion
blockquote Sirve para hacer una cita bibliografica similar a la etiqueta "q" pero es desplegada como un cuadro Documentacion
cite Sirve para hacer una cita pero se puede redireccionar a ella, esta debe tener un titulo Documentacion
dfn Es usado para marcar la palabra que va a ser definida posteriormente Documentacion
abbr Es usado para marcar que una cadena es una abreviacion o acronimo Documentacion
small Se usa para desplegar el texto en un tamaño de fuente mas pequeño Documentacion
ins Es usado para resaltar contenido que fue añadido recientemente al html Documentacion
del Es usado para resaltar contenido que sera eliminado recientemente al html Documentacion
code Se usa para desplegar un fragmento en formato de codigo, debe estar dentro el elemento pre, ya que code solo es una linea Documentacion
pre Sirve para renderizar el fragmento interior en el mismo formato que fue ingresado Documentacion
samp Es usado para marcar el texto contenido como un output de la maquina Documentacion
kbd La etiquta despliega el texto contenido indicando que se espera una entrada de teclado Documentacion
address Se usa para indicar que el contenido es una informacion de contacto Documentacion
Special Chars Es una secuencia de caracteres que permite ingresar caracteres como ser & < > Documentacion
