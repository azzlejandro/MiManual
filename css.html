Cascading Style Sheets

El código CSS hace la vida más fácil al desarrollador front-end al separar las estructura de un documento HTML de su presentación. Dicho de otro modo el HTML actuaría como es esqueleto de la web, definiendo su estructura básica, y el CSS añadiría toda la capa de personalización sobre el que la web define su aspecto final. Siguiendo este fundamento resulta muy fácil para un diseñador web realizar cambios en la apariencia de una web sin afectar de manera dramática a su contenido. El contenido siempre será el mismo, solo cambia como aquello que podemos ver. CSS es fácil de entender y aprender, y nos da un potente control de cómo diseñar los documentos HTML.

Colores en Css

Modelo Descripcion sintaxis
RGB Se estabece un color en base a las intensidades de los colores primarios. Tambien se admite un parametro adicional que es la transparencia rgb(red,green,blue,opacity) e.j rgb(112,60,143,0.2)
Valor Hexadecimal Es un valor de 6 digitos donde el primero indica el valor del rojo, el segundo del verde y el ultimo del azul e.j #703c8f
HSL El modelo admite 3 parametros hue(Tono,[0-360]),saturation (Saturacion,[0-100]%) lightness(luminosidad,[0-100]%) en este modelo tambien es posible agregar el parametro opacidad HSL(278,41%,40%,0.2)

Selectores en Css

Modelo Descripcion sintaxis
Selector Universal Se usa el * para seleccionar todos los elementos y aplicar el estilo *{color:red;}
Selector de tipo Se usa la etiqueta del elemento y se aplica el estilo a dicho elemento p{color:green;}
Selector de clase Se selecciona los elementos en base a su atributo class y se les aplica los elementos .clase1{color:blue;}
Selector de ID Se selecciona los elementos en base a su atributo id y se les aplica los elementos #ID1{color:blue;}
Selector de atributos Se selecciona los elementos que tengan un atributo dado td[att]{color:red;}
Se selecciona los elementos que el atributo seleccionado sea igual a un valor td[att=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado contenga un valor dado td[att ~=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado donde el primer dato del atributo sea el valor dado td[att ^=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado donde el ultimo dato del atributo sea el valor dado td[att $=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado donde alguno de los datos del atributo sea el valor dado td[att *=value]{color:red;}
Selector de pseudo clases Permite seleccionar elementos en base a su estado como ser posicion o interaccion button:hover{...}; button:disabled{...};
Selector de pseudo elementos Permite seleccionar partes de un elemento para aplicar un estilo p::first-line{...}
Union de selectores Realiza varias selecciones a las que se les aplicara un solo estilo Sel1,Sel2{...}
Seleccion de elementos descendientes Se aplica el estilo a elementos descendientes de otros (no necesariamente hijos) Abuelo nieto{...}
Seleccion de elementos hijos Se aplica el estilo a elementos hijos de otros Padre>hijo{...}
Seleccion de elementos hermanos Se aplica el estilo a elementos que son hermanos hermano1~hermano2{...}
Seleccion de elementos hermanos adyacentes Se aplica el estilo a elementos que son hermanos y estan inmediatamente despues hermano1+hermano2{...}
