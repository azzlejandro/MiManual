Colores en CSS
Modelo Descripcion Sintaxis
RGB Se estabece un color en base a las intensidades de los colores primarios. Tambien se admite un parametro adicional que es la transparencia rgb(red,green,blue,opacity) e.j rgb(112,60,143,0.2)
Valor Hexadecimal Es un valor de 6 digitos donde el primero indica el valor del rojo, el segundo del verde y el ultimo del azul e.j #703c8f
HSL El modelo admite 3 parametros hue(Tono,[0-360]),saturation (Saturacion,[0-100]%) lightness(luminosidad,[0-100]%) en este modelo tambien es posible agregar el parametro opacidad HSL(278,41%,40%,0.2)
Selectores en CSS
Selector Descripcion Ejemplo
Selector Universal Se usa el * para seleccionar todos los elementos y aplicar el estilo *{color:red;}
Selector de tipo Se usa la etiqueta del elemento y se aplica el estilo a dicho elemento p{color:green;}
Selector de clase Se selecciona los elementos en base a su atributo class y se les aplica los elementos .clase1{color:blue;}
Selector de ID Se selecciona los elementos en base a su atributo id y se les aplica los elementos #ID1{color:blue;}
Selector de atributos Se selecciona los elementos que tengan un atributo dado td[att]{color:red;}
Se selecciona los elementos que el atributo seleccionado sea igual a un valor td[att=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado contenga un valor dado td[att ~=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado donde el primer dato del atributo sea el valor dado td[att ^=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado donde el ultimo dato del atributo sea el valor dado td[att $=value]{color:red;}
Se selecciona los elementos que el atributo seleccionado donde alguno de los datos del atributo sea el valor dado td[att *=value]{color:red;}
Selector de pseudo clases Permite seleccionar elementos en base a su estado como ser posicion o interaccion button:hover{...}; button:disabled{...};
Selector de pseudo elementos Permite seleccionar partes de un elemento para aplicar un estilo p::first-line{...}
Union de selectores Realiza varias selecciones a las que se les aplicara un solo estilo Sel1,Sel2{...}
Seleccion de elementos descendientes Se aplica el estilo a elementos descendientes de otros (no necesariamente hijos) Abuelo nieto{...}
Seleccion de elementos hijos Se aplica el estilo a elementos hijos de otros Padre>hijo{...}
Seleccion de elementos hermanos Se aplica el estilo a elementos que son hermanos hermano1~hermano2{...}
Seleccion de elementos hermanos adyacentes Se aplica el estilo a elementos que son hermanos y estan inmediatamente despues hermano1+hermano2{...}

Especificidad

La Especificidad es lo que permite determinar que estilo se va aplicar a un elemento, ya que pueden haber varios selectores para un mismo elemento.

La regla consiste en que hay tres niveles para determinar el que gana la especificidad, el id, la clase y el elemento

Si un primer selector tiene mas especificidad en el campo de id da igual que el segundo tenga mas especifidad en los otros dos campos, el primero ya gano la especificidad

Estilos en linea

Los estilos en linea son declaraciones css usando el atributo style de un elemento HTML, este tiene prioridad sobre todas las otras declaraciones y no toma en cuenta la especifidad

!Important

El uso de !important sirve para darle prioridad a ciertas declaraciones de CSS, estas van por encima de los estilos aplicados usando el atributo style

Orden

Cuando tenemos varios selectores que tienen la misma espeficidad el selector que este al ultimo sera el que tenga la prioridad

Herencia

Algunos de los elementos CSS heredan las propiedades de sus padres pero otros no, entonces para indicar que una propiedad se debe heredar del padre se usa la sentencia "inherit"

Si se quisiera romper la herencia se usa la sentencia "initial"

Modelo de Cajas

Cada elemento es considerado como una caja y se compone por

*Contenido
Es el area donde se encuentra el texto, imagen etc, este es afectado por:
width
min-width
max-width
height
min-height
max-height
*Relleno
Es el area de relleno del elemento, este es afectado por:
padding
padding-top
padding-right
padding-bottom
padding-left
*Borde
Es el borde del elemento, puede tener un grosor y color que estan determinados por:
border
border-width
border-style
border-color
*Margen
Es el margen del elemento, es usado para separar al elemento de sus vecinos:
margin
margin-top
margin-right
margin-bottom
margin-left

Colapsamiento de margenes verticales

Los margenes verticales en HTML se solapan hasta que uno choque con el borde o relleno del otro elemento, por tanto la distancia entre estos sera igual a la del margen mas grande

FlexBox

La propiedad display flex permite establecer contenedores flexibles los cuales agrupan los componentes a lo largo de un eje principal y las lineas o ejes transversales agrupan las lineas flexibles


Tamaños

Otra caracteristica de flexbox es que su tamaño principal es en base a su ancho si es rows y alto si son columns, de manera similar si el tamaño principal es el alto, el tamaño transversal sera el ancho

Ejes principales

Es posible escoger el eje principal usando la propiedad flex-direction, las opciones son

  • rows
  • rows-reverse
  • columns
  • columns-reverse

Colocar elementos en el eje principal

Se puede establecer como se distribuyen los elementos en el eje principal usando la propiedad justify-content, esta puede recibir:

  • flex-start
  • flex-end
  • space-between
  • center
  • space-around

Colocar elementos en el eje transversal

Se puede establecer como se distribuyen los elementos en el eje transversal usando la propiedad align-items, esta puede recibir:

  • flex-start
  • flex-end
  • stretch
  • center

Flex-wrap

La propiedad flex-wrap se encarga de definir si el contenedor tiene una o varias lineas flexibles y ademas determina la direccion de estas, esta propiedad recibe

  • nowrap
  • wrap
  • wrap-reverse
